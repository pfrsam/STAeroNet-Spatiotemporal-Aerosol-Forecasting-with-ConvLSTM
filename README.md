# STAeroNet-Spatiotemporal-Aerosol-Forecasting-with-ConvLSTM
Deep learning project for spatiotemporal Aerosol Optical Depth (AOD) forecasting using satellite imagery and ConvLSTM. This project processes MODIS MAIAC AOD GeoTIFF data, performs preprocessing and sequence generation, and trains a ConvLSTM-based model to predict future air pollution patterns over Korea.
